## [1.0.0] - {PR_MERGE_DATE}

### Added

- **Organize Files** command with interactive folder picker and AI-powered categorization
- **Quick Tidy Downloads** command for one-click organization of Downloads folder
- **View Organization History** command with undo support
- Custom settings form for AI model selection and configuration
- Automatic OpenCode server management (connects to existing or spawns new)
- Category-based proposal grouping with smart action suggestions
- Toast notifications with inline undo actions
